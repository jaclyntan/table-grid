@mdo/table-grid

This is a serious-fun-dumb-useful experiment for writing a simple, responsive, and mobile-first CSS grid system. With tables.

Sass variables

Only 4 variables needed:
  • mobile-breakpoint: 600px;
  • $container-width: 1000px;
  • $gutter-width: 50px;
  • $column-count: 13;
